# 6\-10岁少儿课后学习小程序 开发需求文档（Word精简版）

# 一、项目概述

项目名称：6\-10岁少儿课后学习小程序（适配1\-4年级小学生）

核心定位：专注课后巩固、口算练习、单词背诵、每日打卡，轻量化、易操作，适配家长监督、孩子自主学习，微信小程序专用（无需开发电脑网页）

核心目标：低成本上线、高留存、可变现，满足小学生课后碎片化学习，家长可实时监督

# 二、核心功能需求（必做，外包开发重点）

## （一）基础登录\&amp;适配

- 微信一键登录，无需注册，自动绑定微信账号，支持切换账号

- 适配手机、平板双端，界面适配儿童操作（按钮偏大、字体清晰）

- 护眼模式（可手动开启，降低屏幕亮度、减少蓝光）、深色模式，答题音效可开关

- 数据云端保存，换设备登录不丢失学习记录、错题、打卡数据

## （二）首页核心模块

- 年级选择（1\-4年级），选择后自动匹配对应年级题库、单词表、学习内容

- 每日学习任务推送（语文\+数学\+英语各1组基础题，自动更新）
  - 任务卡片展示：每个任务显示科目图标、名称、描述、进度条、完成数量/总数量
  - 进度状态：未开始（0%进度）、进行中（部分完成）、已完成（100%进度）
  - 点击交互：
    - 未完成任务：点击直接跳转到对应科目练习页面
    - 已完成任务：点击弹出对话框询问"是否继续练习"，确认后跳转
    - 跳转路径：语文→语文课后巩固页、数学→口算练习页、英语→单词背诵页
  - 视觉反馈：已完成任务显示绿色背景、"✓ 已完成"徽章、绿色对勾图标
  - 初始数据设置：语文练习初始展示0/5、8/15、8/8三种状态，方便测试显示效果
  - 组件实现：
    - `DailyTask.vue` - 每日任务卡片组件
    - 集成状态管理，实时更新任务完成度
    - 支持任务点击跳转到对应练习页面

- 打卡入口：显示连续打卡天数、当前打卡进度，打卡勋章展示

- 快捷入口：口算练习、单词背诵、语文巩固、错题本、学习报告

- 排行榜入口（班级/全网，按学习时长、答题正确率排序）

- 家长中心入口（单独入口，需验证后进入）

## （三）数学专项：口算练习

- 题库：1\-4年级校内同步，涵盖加减乘除、进退位、混合运算、单位换算，按基础/进阶/挑战分级

- 答题模式：随机出题，支持限时闯关（1分钟/3分钟）、自由练习

- 批改功能：自动批改，秒出对错，错题实时标记并显示正确答案

- 错题管理：自动归集口算错题，支持错题重做、举一反三同类题推送

- 打卡关联：完成每日口算练习可点亮当日打卡，达标解锁积分/勋章

## （四）英语专项：单词背诵

- 单词库：1\-4年级校内同步（人教/外研通用），按单元分类，支持版本切换

- 背诵功能：单词跟读（标准发音）、听力听写、看图认词、中英互译、拼写闯关

- 复习机制：按艾宾浩斯遗忘曲线推送复习单词，强化记忆

- 进度管理：显示单词背诵进度、掌握程度评分（已掌握/待复习/未掌握）

- 附加：简单短语、短句日积月累，贴合校内课本

## （五）语文专项：课后巩固

- 核心功能：生字听写、拼音认读、组词造句，同步校内课本生字
  - 生字听写：根据拼音写汉字，提供即时反馈
  - 拼音认读：根据汉字写拼音，每题即时判断
  - 组词造句：用给定词语造句，可编辑提交
  - 年级适配：1-2年级每次5题，3-4年级每次8题

- 拓展功能：古诗词背诵、默写闯关，自动批改默写对错
  - 古诗词展示：显示古诗全文、作者、时代背景
  - 背诵模式：点击确认完成后记录进度
  - 默写模式：提示在纸上书写，完成后确认

- 补充功能：课外阅读短篇答题、语文常识小练习，每日1组
  - 阅读理解：阅读短文后回答选择题
  - 常识答题：趣味语文知识小测试

### 技术实现说明
- **组件架构**：
  - `DictationPractice.vue` - 生字听写练习组件
  - `PinyinPractice.vue` - 拼音认读练习组件
  - `WordSentencePractice.vue` - 组词造句练习组件
  - `PoemRecitePractice.vue` - 古诗背诵练习组件
  - `PoemWritePractice.vue` - 古诗默写练习组件
  - `OptionPractice.vue` - 选择题通用组件
  - `AudioPlayer.vue` - 音频播放器组件
  - `PracticeHeader.vue` - 练习页面头部
  - `PracticeProgress.vue` - 练习进度条
  - `FeedbackDisplay.vue` - 答案反馈显示
  - `NavigationButtons.vue` - 导航按钮组件

- **状态管理**：使用Pinia管理练习进度，使用Composables管理反馈状态
- **用户体验**：
  - 提供即时反馈，答对显示鼓励提示，答错显示正确答案
  - 音频播放器简化动画，避免界面抖动
  - 支持导航上一题、下一题，实时更新进度
  - 完成练习后显示结果并记录积分

## （六）每日打卡体系

- 打卡内容：口算\+单词\+语文练习三合一，完成任意2项即可点亮当日打卡

- 打卡奖励：连续打卡可获得积分、专属勋章，支持等级升级（如：启蒙童→进阶童）

- 分享与补签：打卡记录可一键分享至微信群，支持断卡补签（每日1次机会）

- 节假日专属：节假日打卡皮肤、额外积分奖励，提升孩子积极性

## （七）错题本\&amp;学习档案

- 错题本：自动归集所有科目错题，按科目、题型分类，支持搜索、筛选

- 学习分析：每周薄弱项分析、能力短板报告，统计答题正确率、学习时长

- 学习报告：每日/每周/月度报告，清晰展示学习情况，可转发给家长

## （八）家长管理后台

- 监督功能：查看孩子每日做题数量、正确率、学习时长、打卡情况

- 消息通知：打卡提醒、未完成学习通知，支持自定义提醒时间

- 账号管理：绑定孩子账号，支持多孩子账号切换、修改孩子年级

- 报告查看：查看孩子学习周报、月度报告，了解学习进度

## （九）会员\&amp;付费功能（变现核心）

- 免费权限：每日基础题库（少量题目）、基础打卡、简单错题本

- 付费权限：无限刷题、解锁全部单词书/题库、艾宾浩斯复习、错题打印、去除广告

- 套餐设置：月卡29元、季卡69元、年卡199元，支持微信一键支付

- 裂变功能：邀请好友注册，双方各获得3天会员时长

## （十）基础通用功能

- 答题防作弊：设置答题最小间隔，防止乱点秒答

- 意见反馈：家长/孩子可提交反馈，后台可查看

- 必备模块：关于我们、使用说明、隐私政策（用于审核上线）

# 三、开发要求

- 开发类型：微信原生小程序（无需开发APP、电脑网页）

- 上线周期：标准版7\-12天，完整版15\-20天，模板二开3\-5天

- 后台管理：支持题库更新、用户管理、订单管理、数据统计（答题正确率、留存率、付费率）

- 兼容性：适配主流手机、平板及微信最新版本，无闪退、卡顿

- 审核支持：提供所需资料，协助小程序审核上线

# 四、后期维护要求

- 服务器：提供1年服务器使用，后期可续费（800\-1200元/年）

- bug修复：上线后3个月内，免费修复小程序bug

- 题库更新：协助提供基础题库，支持后期自主更新

# 五、Trae IDE \+ 火山引擎豆包模型 简易开发流程（适配本需求）

## （一）前期准备（1\-2天）

- 工具准备：安装Trae IDE（国内版，默认搭载豆包1\.5\-pro模型）、注册微信小程序开发者账号（开通支付权限）

- 素材准备：整理1\-4年级校内同步题库、单词表（可由豆包辅助生成基础素材）

- 需求导入：将本需求文档复制到豆包，由豆包解析需求、梳理逻辑，生成开发蓝图（含模块关联、接口清单）

## （二）核心开发（标准版5\-10天）

1. 项目搭建：打开Trae IDE，选择微信小程序模板，借助豆包生成的开发蓝图自动搭建项目架构（无需手动编写基础代码）

2. 功能编码：调用豆包能力实现微信登录、双端适配等基础功能；通过Trae开发口算、单词等核心模块，豆包辅助调试逻辑；开发会员、家长模块并对接微信支付；导入题库并通过Trae配置分类、分级及自动推送。

## （三）测试调试（1\-2天）

- 功能测试：利用Trae内置调试工具检测各模块功能，豆包辅助排查bug、优化交互

- 兼容性测试：适配不同手机、平板及微信版本，确保无闪退、卡顿，界面显示正常

## （四）审核上线（1\-2天）

- 资料准备：豆包辅助生成小程序审核所需资料（隐私政策、使用说明等）

- 提交审核：通过微信开发者平台提交小程序，Trae协助整理审核材料

- 上线部署：审核通过后，通过Trae完成服务器部署，同步题库、测试数据后正式上线

## （五）后期维护

- 借助Trae后台管理题库、用户数据，豆包辅助优化功能、修复bug

- 按周期更新题库，通过Trae一键同步，无需重新部署

# 七、数据云端保存（fullstack\-developer Skill 实现详解）

## （一）具体实现步骤（Trae自动执行）

调用fullstack\-developer Skill后，自动完成以下操作：

1. 自动创建云端数据库及用户表、错题表等适配数据表，无需手动设计。

2. 自动开发数据上传（操作后同步至云端）、拉取（登录后同步历史数据）接口，无需手动编码。

3. 自动配置接口加密，适配微信小程序权限，契合隐私政策，避免数据泄露。

4. 自动添加异常兼容逻辑，网络不佳时本地缓存，恢复后自动同步，确保数据不丢失。

## （二）服务器相关说明

无需手动配置服务器，全程由Trae IDE \+ fullstack\-developer Skill自动完成：

- 无需手动购买、搭建服务器，Trae自动分配适配小程序的云端服务器存储数据。

- 无需手动配置服务器环境，Skill自动分配合适资源，无需调试参数。

- 后期服务器运行、续费、故障修复由Trae协助完成，无需自行管理。

## （三）与原有功能衔接说明

该功能直接衔接微信登录，用户登录后自动绑定云端账号，实现“登录即同步数据”，换设备可正常拉取历史数据，与现有话术完全适配，直接复制即可触发。
